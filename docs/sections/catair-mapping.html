Mapping to Legacy Systems

Verfiable Credentials are often formed from, or delivered to systems which do not support JSON-LD.

For example:

  • ACE Automated Broker Interface (ABI) CBP and Trade Automated Interface Requirements (CATAIR)

This table shows how key data elements of a Mill Test Report Credential can be related to existing CATAIR documents, and how certain data elements might be new, and will have no mapping to CATAIR.

JSON Pointer Value CATAIR
/type/1 MillTestReportCredential ❌ Absent
/issuer/location/address/streetAddress 701 Brazos St ✅ Present
/credentialSubject/inspections/0/heatNumber 841 ❌ Absent
